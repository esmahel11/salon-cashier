# 📱 فێرکاری خاڵ بە خاڵ بۆ بەکارهێنانی ئەپەکە لەسەر مۆبایل
## بۆ Xiaomi Redmi Pad Pro (و هەموو مۆبایلەکان)

---

## بەشی یەکەم: ئامادەکاری لەسەر کۆمپیوتەر 💻

### خاڵی ١: دامەزراندنی Node.js

**١.١** بڕۆ بۆ ئەم لینکە لە کۆمپیوتەرەکەت:
```
https://nodejs.org
```

**١.٢** دوگمەی سەوز کە نووسراوە **"LTS"** کلیک بکە (ئەمە وەشانی پێشنیارکراوە)

**١.٣** فایلەکە دابەزێنە و دامەزرێنە (وەک هەر پرۆگرامێکی تر)

**١.٤** بۆ دڵنیابوونەوە، Command Prompt بکەرەوە و ئەمە بنووسە:
```bash
node --version
```
ئەگەر ژمارەیەکت پیشان دا (وەک v18.17.0)، واتە سەرکەوتووە! ✅

---

### خاڵی ٢: دامەزراندنی پێداویستیەکانی پرۆژەکە

**٢.١** فۆڵدەری پرۆژەکە بکەرەوە (ئەو فۆڵدەرەی کە تێیدا backend و frontend هەیە)

**٢.٢** لە ناو فۆڵدەرەکە، لەسەر شوێنێکی بەتاڵ **Shift + Right Click** بکە

**٢.٣** هەڵبژێرە **"Open PowerShell window here"** یان **"Open command window here"**

**٢.٤** ئەم فەرمانانە بە ڕیزبەندی بنووسە:

```bash
cd backend
npm install
```
چاوەڕوان بە تا تەواو بێت (دەتوانێت ٢-٣ خولەک بخایەنێت)

**٢.٥** پاشان ئەمە بنووسە:
```bash
cd ..
cd frontend
npm install
```
دیسان چاوەڕوان بە (دەتوانێت ٣-٥ خولەک بخایەنێت)

**٢.٦** کاتێک تەواوبوو، ئەم پەیامە دەبینیت:
```
added XXX packages
```
واتە سەرکەوتووە! ✅

---

### خاڵی ٣: دروستکردنی ئایکۆنەکان

**٣.١** بڕۆ بۆ فۆڵدەری: `frontend/public/`

**٣.٢** فایلی `generate-icons.html` دووجار کلیک بکە

**٣.٣** لە وێبگەڕەکە دەکرێتەوە و خۆکارانە دوو فایل دادەبەزێت:
- `icon-192.png`
- `icon-512.png`

**٣.٤** ئەم دوو فایلە بگوێزەرەوە بۆ فۆڵدەری `frontend/public/`

## بەشی دووەم: دەستپێکردنی سیستەمەکە 🚀

### خاڵی ٤: کردنەوەی سیستەمەکە

**٤.١** بگەڕێوە بۆ فۆڵدەری سەرەکی پرۆژەکە

**٤.٢** فایلی `start-mobile.bat` دووجار کلیک بکە

**٤.٣** دوو پەنجەرەی Command Prompt دەکرێنەوە:
- یەکێک بۆ Backend (ڕەش)
- یەکێک بۆ Frontend (ڕەش)

**٤.٤** چاوەڕوان بە تا ئەم پەیامانە ببینیت:

لە پەنجەرەی Backend:
```
🚀 سێرڤەر بە سەرکەوتوویی دەستی پێکرد
📱 لینکی مۆبایل: http://192.168.X.X:5000
```

لە پەنجەرەی Frontend:
```
Compiled successfully!
Local: http://localhost:3000
```

**٤.٥** تێبینی بکە لە ژمارەی IP کە پیشان دەدرێت (وەک: 192.168.1.100)
📝 **ئەم ژمارەیە بنووسە یان وێنەیەکی لێ بگرە - پێویستت پێیەتی!**

## بەشی سێیەم: بەستنەوەی مۆبایلەکەت 📱

### خاڵی ٥: دڵنیابوونەوە لە WiFi

**٥.١** لە کۆمپیوتەرەکەت، سەیری ناوی WiFi بکە کە پێوەی بەستراویت

**٥.٢** لە مۆبایلەکەت (Xiaomi Redmi Pad Pro):
- بڕۆ بۆ **Settings** (ڕێکخستنەکان)
- بڕۆ بۆ **Wi-Fi**
- دڵنیابە کە بەستراویت بە **هەمان WiFi** ی کۆمپیوتەرەکە

⚠️ **زۆر گرنگە: هەردووکیان دەبێت لە هەمان WiFi بن!**

---

### خاڵی ٦: کردنەوەی ئەپەکە لە مۆبایل

**٦.١** لە مۆبایلەکەت، **Chrome** بکەرەوە (یان Firefox)

**٦.٢** لە شریتی ناونیشان، ئەم شێوازە بنووسە:
```
http://192.168.X.X:3000
```
(ئەو ژمارە IP یەی کە لە خاڵی ٤.٥ نووسیت بەکاربهێنە)

**نموونە:**
```
http://192.168.1.100:3000
```

**٦.٣** Enter بکە

**٦.٤** ئەگەر هەموو شتێک ڕاست بێت، پەڕەی چوونەژوورەوە دەبینیت! 🎉

---

### خاڵی ٧: چوونەژوورەوە

**٧.١** لە پەڕەی Login:
- **ناوی بەکارهێنەر:** بنووسە `admin`
- **وشەی نهێنی:** بنووسە `admin123`

**٧.٢** دوگمەی **"چوونەژوورەوە"** کلیک بکە

**٧.٣** ئێستا دەچیتە ناو سیستەمەکە! ✅

## بەشی چوارەم: دامەزراندن وەک ئەپ 📲

### خاڵی ٨: دامەزراندنی PWA

**٨.١** کاتێک لە Chrome ئەپەکەت کردەوە، لە سەرەوەی پەڕەکە دەبینیت:
```
"Add [App Name] to Home screen"
```

**٨.٢** ئەگەر ئەم پەیامە نەبینی:
- لە گۆشەی سەرەوەی ڕاست، سێ خاڵەکە (⋮) کلیک بکە
- هەڵبژێرە **"Add to Home screen"** یان **"Install app"**

**٨.٣** پەنجەرەیەک دەردەکەوێت:
- ناوێک بنووسە بۆ ئەپەکە: `کاشێری سالۆن`
- کلیک لە **"Add"** یان **"زیادکردن"** بکە

**٨.٤** ئایکۆنی ئەپەکە لەسەر سکرینی سەرەکی دەردەکەوێت! 🎉

**٨.٥** ئێستا دەتوانیت Chrome دابخەیت و لە ئایکۆنەکە ئەپەکە بکەیتەوە

## بەشی پێنجەم: بەکارهێنانی ئەپەکە 🎯

### خاڵی ٩: ناسینەوەی ڕووکاری مۆبایل

کاتێک ئەپەکە لە مۆبایل دەکەیتەوە، دەبینیت:

**لە سەرەوە:**
- سەردێڕی پەڕەکە
- دوگمەی دەرچوون

**لە ناوەڕاست:**
- ناوەڕۆکی سەرەکی (داشبۆرد، فرۆشتن، هتد)

**لە خوارەوە (ناڤیگەیشنی نوێ):**
- 📊 **سەرەکی** - داشبۆرد
- 💰 **فرۆشتن** - فرۆشتنی بەرهەم
- 📦 **کۆگا** - بەڕێوەبردنی کۆگا
- ✂️ **خزمەتگوزاری** - خزمەتگوزارییەکان
- 👥 **کڕیار** - کڕیارەکان

---

### خاڵی ١٠: بەکارهێنانی تایبەتمەندیەکان

#### فرۆشتنی بەرهەم:

**١٠.١** کلیک لە **💰 فرۆشتن** بکە لە خوارەوە

**١٠.٢** کڕیارێک هەڵبژێرە (یان نوێ زیاد بکە)

**١٠.٣** بەرهەمێک هەڵبژێرە

**١٠.٤** بڕەکە دیاری بکە

**١٠.٥** کلیک لە **"زیادکردن بۆ سەبەتە"** بکە

**١٠.٦** ئەگەر داشکاندنت هەیە، بینووسە

**١٠.٧** کلیک لە **"تەواوکردنی فرۆشتن"** بکە

**١٠.٨** پسوڵەکە پیشان دەدرێت - دەتوانیت پرینتی بکەیت

#### زیادکردنی بەرهەم بۆ کۆگا:

**١٠.٩** کلیک لە **📦 کۆگا** بکە

**١٠.١٠** کلیک لە دوگمەی **"+ زیادکردنی مادە"** بکە

**١٠.١١** زانیاریەکان پڕبکەرەوە:
- ناوی مادە
- بارکۆد
- بڕ
- نرخی کڕین
- نرخی فرۆشتن

**١٠.١٢** کلیک لە **"پاشەکەوتکردن"** بکە

## بەشی شەشەم: چارەسەرکردنی کێشەکان 🔧

### کێشە ١: ناتوانم ئەپەکە لە مۆبایل بکەمەوە

**چارەسەر:**

✅ **پشکنینی ١:** دڵنیابە هەردووکیان لە هەمان WiFi ن
- لە کۆمپیوتەر: سەیری ناوی WiFi بکە
- لە مۆبایل: سەیری ناوی WiFi بکە
- دەبێت **هەمان ناو** بێت

✅ **پشکنینی ٢:** IP Address ڕاست بەکاربهێنە
- لە کۆمپیوتەر، Command Prompt بکەرەوە
- بنووسە: `ipconfig`
- بگەڕێ بۆ **"IPv4 Address"**
- ئەو ژمارەیە بەکاربهێنە (وەک: 192.168.1.100)

✅ **پشکنینی ٣:** Firewall پشکنین بکە
- لە کۆمپیوتەر، بڕۆ بۆ **Control Panel**
- بڕۆ بۆ **Windows Defender Firewall**
- کلیک لە **"Allow an app through firewall"**
- دڵنیابە کە **Node.js** تیک لێدراوە

✅ **پشکنینی ٤:** سیستەمەکە کارە؟
- لە کۆمپیوتەر، لە وێبگەڕ بڕۆ بۆ: `http://localhost:3000`
- ئەگەر کاری کرد، واتە سیستەمەکە باشە
- کێشەکە لە بەستنەوەی مۆبایلەوەیە

---

### کێشە ٢: ئایکۆنەکان دەرناکەون

**چارەسەر:**

**٢.١** بڕۆ بۆ: `frontend/public/`

**٢.٢** فایلی `generate-icons.html` دووجار کلیک بکە

**٢.٣** دوو فایل دادەبەزێت - بیانگوێزەرەوە بۆ هەمان فۆڵدەر

**٢.٤** سیستەمەکە دابخە و دیسان دەستی پێبکەرەوە

---

### کێشە ٣: سیستەمەکە خاو دەبێتەوە

**چارەسەر:**

**٣.١** دڵنیابە کە دوو پەنجەرەی Command Prompt کراون (Backend و Frontend)

**٣.٢** ئەگەر یەکێکیان داخرا، `start-mobile.bat` دیسان بکەرەوە

**٣.٣** ئەگەر هەڵەیەک دەبینیت، وێنەیەکی لێ بگرە و بیردەوە

---

### کێشە ٤: زۆر خاوە

**چارەسەر:**

**٤.١** دڵنیابە کە WiFi خێرایە

**٤.٢** ئەپەکان لەسەر مۆبایل دابخە (بەتایبەت ئەوانەی ڤیدیۆ دەبینن)

**٤.٣** کۆمپیوتەرەکە نزیکتر بکەرەوە لە Router

## بەشی حەوتەم: تێبینیە گرنگەکان ⚠️

### ١. کۆمپیوتەرەکە دەبێت کراو بێت
- بۆ ئەوەی لە مۆبایل کار بکات، کۆمپیوتەرەکە دەبێت کراو بێت
- سیستەمەکە (Backend و Frontend) دەبێت کاربکات

### ٢. هەمان WiFi
- هەمیشە دڵنیابە کە هەردووکیان لە هەمان WiFi ن

### ٣. IP Address دەگۆڕێت
- ئەگەر کۆمپیوتەرەکە دووبارە دەستت پێکردەوە، IP Address دەتوانێت بگۆڕێت
- ئەگەر ئەپەکە کاری نەکرد، IP نوێ بدۆزەرەوە

### ٤. پاشەکەوتکردنی داتا
- هەموو داتاکان لە بنکەی داتای کۆمپیوتەرەکە پاشەکەوت دەکرێن
- ئەگەر کۆمپیوتەرەکە بشکێت، داتاکان دەمێننەوە

## بەشی هەشتەم: ڕاوێژکاری بۆ بەکارهێنانی باش 💡

### ١. بەکارهێنانی ئۆفلاین
- دوای دامەزراندنی PWA، هەندێک بەش بەبێ ئینتەرنێت کار دەکات
- بەڵام بۆ پاشەکەوتکردنی داتا، پێویستە بەستنەوەت هەبێت

### ٢. خێراکردنەوە
- ئەپەکە لە جاری یەکەم کەمێک خاو دەبێت
- دوای ئەوە خێراتر دەبێت چونکە کاش دەکرێت

### ٣. نوێکردنەوە
- ئەگەر گۆڕانکاریەکت کرد لە کۆد، دەبێت ئەپەکە نوێ بکەیتەوە
- لە Chrome، مینیۆ بکەرەوە و **"Refresh"** بکە

### ٤. چاککردنی کێشەکان
- ئەگەر شتێک کاری نەکرد، ئەپەکە دابخە و دیسان بیکەرەوە
- ئەگەر دیسان کاری نەکرد، کۆمپیوتەرەکە Restart بکە
