# 🚀 دەست پێبکە لێرەوە

## خاڵی ١: دامەزراندنی پێداویستیەکان (یەک جار تەنها)

### ١.١ دامەزراندنی Backend

لە Command Prompt یان PowerShell:

```bash
cd backend
npm install
```

چاوەڕوان بە تا تەواو بێت (٢-٣ خولەک)

### ١.٢ دامەزراندنی Frontend

```bash
cd ..
cd frontend
npm install
```

چاوەڕوان بە تا تەواو بێت (٣-٥ خولەک)

---

## خاڵی ٢: دروستکردنی ئایکۆنەکان

### ٢.١ بڕۆ بۆ فۆڵدەری:
```
frontend/public/
```

### ٢.٢ فایلی `generate-icons.html` دووجار کلیک بکە

### ٢.٣ دوو فایل دادەبەزێت:
- icon-192.png
- icon-512.png

### ٢.٤ ئەم دوو فایلە بگوێزەرەوە بۆ هەمان فۆڵدەر (`frontend/public/`)

---

## خاڵی ٣: دەستپێکردن بۆ بەکارهێنان لەسەر مۆبایل

### ٣.١ فایلی `start-mobile.bat` دووجار کلیک بکە

### ٣.٢ دوو پەنجەرەی Command Prompt دەکرێنەوە

### ٣.٣ چاوەڕوان بە تا ئەم پەیامانە ببینیت:

**لە پەنجەرەی یەکەم (Backend):**
```
🚀 سێرڤەر بە سەرکەوتوویی دەستی پێکرد
📱 لینکی مۆبایل: http://192.168.X.X:5000
```

**لە پەنجەرەی دووەم (Frontend):**
```
Compiled successfully!
```

### ٣.٤ تێبینی بکە لە ژمارەی IP (وەک: 192.168.1.100)

---

## خاڵی ٤: کردنەوە لە مۆبایل

### ٤.١ دڵنیابە کە مۆبایل و کۆمپیوتەر لە هەمان WiFi ن

### ٤.٢ لە مۆبایلەکەت Chrome بکەرەوە

### ٤.٣ بڕۆ بۆ:
```
http://[IP-ی-کۆمپیوتەرەکەت]:3000
```

نموونە:
```
http://192.168.1.100:3000
```

### ٤.٤ پەڕەی Login دەبینیت!
