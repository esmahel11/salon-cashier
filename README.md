# 🎀 سیستەمی کاشێری ئارایشتگای خانمان
## Beauty Salon POS System

سیستەمێکی تەواوی کاشێر بۆ ئارایشتگاکان لەگەڵ بەڕێوەبردنی کۆگا، فرۆشتن، کڕیارەکان، چاوپێکەوتنەکان و ڕاپۆرتەکان.

---

## 🌟 تایبەتمەندییەکان

### تایبەتمەندیەکانی سەرەکی
✅ سیستەمی چوونەژوورەوە (Admin & Employee)  
✅ بەڕێوەبردنی کۆگای مادەکان  
✅ سیستەمی فرۆشتن لەگەڵ داشکاندن  
✅ بەڕێوەبردنی خزمەتگوزارییەکان  
✅ بەڕێوەبردنی کڕیارەکان  
✅ سیستەمی حەجزکردنی چاوپێکەوتن  
✅ تۆماری خەرجییەکان  
✅ ڕاپۆرتە جۆراوجۆرەکان  
✅ پرینتکردنی پسوڵە  
✅ ئاگاداری مادە کەمەکان  
✅ دوو زمان (کوردی و عەرەبی)  

### 📱 تایبەتمەندیەکانی نوێ بۆ مۆبایل
✅ دیزاینی تەواو ڕێسپۆنسیڤ بۆ مۆبایل و تابلێت  
✅ ناڤیگەیشنی خوارەوە بۆ ئاسانکاری لە گەشتکردن  
✅ دوگمەی گەورەتر بۆ تاچ  
✅ Progressive Web App (PWA) - دامەزراندن وەک ئەپ  
✅ کارکردنی ئۆفلاین  
✅ گونجاو لەگەڵ Xiaomi Redmi Pad Pro و هەموو مۆبایلەکان

## 🛠️ تەکنەلۆژیاکان

**Backend:**
- Node.js
- Express.js
- SQLite (better-sqlite3)
- JWT Authentication
- bcryptjs

**Frontend:**
- React.js 18
- React Router v6
- Axios
- React-to-Print
- PWA Support

---

## 📋 پێداویستییەکان

- Node.js (v14 یان بەرزتر)
- npm یان yarn

## 🚀 دامەزراندن و هەڵگرتن

### دامەزراندن

```bash
# دامەزراندنی Backend
cd backend
npm install

# دامەزراندنی Frontend
cd ../frontend
npm install
```

### کردنەوە

#### بۆ بەکارهێنان لەسەر کۆمپیوتەر:
```bash
# دووجار کلیک لەسەر start.bat بکە
start.bat
```

#### 📱 بۆ بەکارهێنان لەسەر مۆبایل:
```bash
# دووجار کلیک لەسەر start-mobile.bat بکە
start-mobile.bat
```

### ڕێنماییەکانی مۆبایل

بۆ ڕێنمایی تەواوی دامەزراندن و بەکارهێنان لەسەر مۆبایل:
- 📖 [ڕێنمای خێرا (کوردی)](./QUICK_START_KU.md)
- 📚 [ڕێنمای تەواو (کوردی)](./MOBILE_SETUP_KU.md)

## 📱 چۆنیەتی دامەزراندن وەک ئەپ لەسەر مۆبایل

1. سیستەمەکە بە `start-mobile.bat` دەستپێبکە
2. لە مۆبایلەکەت Chrome بکەرەوە
3. بڕۆ بۆ لینکەکە کە پیشان دەدرێت
4. لە مینیۆی Chrome، هەڵبژێرە "Add to Home screen"
5. ئێستا وەک ئەپێکی ڕاستەقینە کار دەکات! 🎉

## 🔧 چارەسەرکردنی کێشەکان

### کێشە: ناتوانم لە مۆبایل بیکەمەوە
- دڵنیابە کە مۆبایل و کۆمپیوتەر لە هەمان WiFi ن
- Firewall ی Windows پشکنین بکە
- IP Address ڕاست بەکاربهێنە

### کێشە: ئایکۆنەکانی PWA دەرناکەون
- فایلی `frontend/public/generate-icons.html` بکەرەوە
- ئایکۆنەکان دابەزێنە و بیانگوێزەرەوە بۆ `frontend/public/`

### کێشە: پۆرتەکان قەفڵن
```bash
# Windows
netstat -ano | findstr :3000
netstat -ano | findstr :5000
taskkill /PID [PID_NUMBER] /F
```
