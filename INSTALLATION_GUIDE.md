# 📖 ڕێنمای تەواوی دامەزراندن و هەڵگرتن
## Beauty Salon POS System - Installation Guide

---

## 📋 پێش دامەزراندن

### پێداویستییەکان:
1. **Node.js** (v14 یان بەرزتر)
   - دابەزاندن لە: https://nodejs.org/
   - پشکنینی دامەزراندن: `node --version`

2. **npm** (دێت لەگەڵ Node.js)
   - پشکنینی: `npm --version`

---

## 🚀 دامەزراندن (هەنگاو بە هەنگاو)

### شێوازی ١: بەکارهێنانی فایلی ئۆتۆماتیک ⭐ (پێشنیارکراو)

**١. کۆپیکردنی پرۆژە:**
- هەموو فۆڵدەرەکان و فایلەکان کۆپی بکە بۆ ناو فۆڵدەرێک بە ناوی `beauty-salon-system`

**٢. دابڵچکە دوو جار لەسەر `start.bat`**
- ئەم فایلە هەموو کارەکان بە ئۆتۆماتیک دەکات:
  - دامەزراندنی پاکەتەکانی Backend
  - دامەزراندنی پاکەتەکانی Frontend
  - دروستکردنی داتابەیس
  - هەڵگرتنی Backend لەسەر پۆرتی 5000
  - هەڵگرتنی Frontend لەسەر پۆرتی 3000

**٣. چاوەڕوانبە تا وێبگەڕەکە خۆکارانە بکرێتەوە**
- ئەگەر خۆکارانە نەکرایەوە، بڕۆ بۆ: `http://localhost:3000`

---

### شێوازی ٢: بە دەستی (Manual)

#### هەنگاوی ١: دامەزراندنی Backend

```bash
# بڕۆ بۆ فۆڵدەری backend
cd backend

# دامەزراندنی پاکەتەکان
npm install

# هەڵگرتنی backend
npm start
