# ڕێنمای دامەزراندنی ئەپی مۆبایل
## سیستەمی کاشێری سالۆنی جوانکاری

### چۆنیەتی دامەزراندن لەسەر Xiaomi Redmi Pad Pro

#### هەنگاوی یەکەم: دامەزراندنی سیستەمەکە

1. **دامەزراندنی Node.js**
   - Node.js دابەزێنە لە: https://nodejs.org
   - وەشانی LTS هەڵبژێرە

2. **کردنەوەی پرۆژەکە**
   ```bash
   # چوونە ناو فۆڵدەری پرۆژەکە
   cd [ناوی-فۆڵدەری-پرۆژەکە]
   
   # دامەزراندنی پێداویستیەکان بۆ Backend
   cd backend
   npm install
   
   # دامەزراندنی پێداویستیەکان بۆ Frontend
   cd ../frontend
   npm install
   ```

3. **دروستکردنی ئایکۆنەکان**
   - فایلی `frontend/public/generate-icons.html` بکەرەوە لە وێبگەڕەکەت
   - دوو فایل دادەبەزێت: `icon-192.png` و `icon-512.png`
   - ئەم دوو فایلە بگوێزەرەوە بۆ `frontend/public/`

#### هەنگاوی دووەم: کردنەوەی سیستەمەکە

1. **کردنەوەی Backend**
   ```bash
   cd backend
   npm start
   ```
   Backend لەسەر `http://localhost:5000` دەکرێتەوە

2. **کردنەوەی Frontend (لە تەرمیناڵێکی نوێ)**
   ```bash
   cd frontend
   npm start
   ```
   Frontend لەسەر `http://localhost:3000` دەکرێتەوە

#### هەنگاوی سێیەم: بەکارهێنان لەسەر مۆبایل

##### ڕێگای یەکەم: لە ڕێگەی WiFi (پێشنیارکراو)

1. **دۆزینەوەی IP Address ی کۆمپیوتەرەکەت**
   
   لە Windows:
   ```bash
   ipconfig
   ```
   بگەڕێ بۆ `IPv4 Address` (وەک: 192.168.1.100)

2. **گۆڕینی کۆنفیگی Frontend**
   
   فایلی `frontend/src/services/api.js` بکەرەوە و بیگۆڕە بۆ:
   ```javascript
   const API_URL = 'http://[IP-ی-کۆمپیوتەرەکەت]:5000/api';
   // نموونە: http://192.168.1.100:5000/api
   ```

3. **دەستپێکردنەوەی Frontend**
   ```bash
   cd frontend
   npm start
   ```

4. **کردنەوە لە مۆبایل**
   - دڵنیابە کە مۆبایلەکەت لە هەمان WiFi یە
   - لە Chrome یان Firefox کردنەوە: `http://[IP-ی-کۆمپیوتەرەکەت]:3000`
   - نموونە: `http://192.168.1.100:3000`

##### ڕێگای دووەم: دامەزراندن وەک ئەپ (PWA)

1. **کردنەوەی سایتەکە لە Chrome**
   - بڕۆ بۆ `http://[IP-ی-کۆمپیوتەرەکەت]:3000`

2. **دامەزراندنی ئەپەکە**
   - لە مینیۆی Chrome (سێ خاڵەکە) کلیک بکە
   - هەڵبژێرە "Add to Home screen" یان "دامەزراندن"
   - ناوێک بۆ ئەپەکە بنووسە: "کاشێری سالۆن"
   - کلیک لە "Add" بکە

3. **بەکارهێنانی ئەپەکە**
   - ئایکۆنی ئەپەکە لەسەر سکرینی سەرەکی دەردەکەوێت
   - کلیک لەسەر بکە بۆ کردنەوە
   - وەک ئەپێکی ڕاستەقینە کار دەکات!

#### تایبەتمەندیەکانی نوێ بۆ مۆبایل

✅ **ناڤیگەیشنی خوارەوە**: مینیۆیەکی خوارەوە بۆ ئاسانکاری لە گەشتکردن
✅ **دیزاینی ڕێسپۆنسیڤ**: هەموو پەڕەکان گونجاون لەگەڵ قەبارەی شاشەی مۆبایل
✅ **دوگمەی گەورەتر**: بۆ ئاسانکاری لە کلیککردن بە پەنجە
✅ **کارکردنی ئۆفلاین**: دوای دامەزراندن، بەبێ ئینتەرنێت کار دەکات
✅ **خێراتر**: Service Worker داتا کاش دەکات بۆ خێراییەکی زیاتر

#### چارەسەرکردنی کێشەکان

**کێشە: ناتوانم لە مۆبایل بیکەمەوە**
- دڵنیابە کە مۆبایل و کۆمپیوتەر لە هەمان WiFi ن
- Firewall ی Windows پشکنین بکە
- IP Address ڕاست بەکاربهێنە

**کێشە: دوگمەکان زۆر بچووکن**
- ئەپەکە وەک PWA دابمەزرێنە
- Zoom ی وێبگەڕەکە بگۆڕە بۆ 100%

**کێشە: ئایکۆنەکان دەرناکەون**
- فایلی `generate-icons.html` بکەرەوە و ئایکۆنەکان دروست بکە
- دڵنیابە کە `icon-192.png` و `icon-512.png` لە `frontend/public/` دان

#### زانیاری زیاتر

- **بەکارهێنەری سەرەتایی**: admin
- **وشەی نهێنی**: admin123
- **پۆرتی Backend**: 5000
- **پۆرتی Frontend**: 3000

#### تێبینیەکان

⚠️ بۆ بەکارهێنانی درێژخایەن، پێشنیار دەکرێت:
1. Backend لەسەر سێرڤەرێک دابنێ (وەک VPS)
2. دۆمەینێک بەکاربهێنە بە SSL
3. ئەپەکە بڵاوبکەرەوە لەسەر هۆستێکی دڵنیا
