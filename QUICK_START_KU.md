# دەستپێکردنی خێرا 🚀

## بۆ بەکارهێنان لەسەر مۆبایل

### هەنگاوی یەکەم: دامەزراندن

```bash
# دامەزراندنی پێداویستیەکان بۆ Backend
cd backend
npm install

# دامەزراندنی پێداویستیەکان بۆ Frontend
cd ../frontend
npm install
```

### هەنگاوی دووەم: دەستپێکردن

#### ڕێگای ئاسان (پێشنیارکراو):
دووجار کلیک لەسەر `start-mobile.bat` بکە

#### ڕێگای دەستی:

**1. دەستپێکردنی Backend:**
```bash
cd backend
npm start
```

**2. دۆزینەوەی IP ی کۆمپیوتەرەکەت:**
```bash
ipconfig
```
بگەڕێ بۆ `IPv4 Address` (نموونە: 192.168.1.100)

**3. دەستپێکردنی Frontend:**
```bash
cd frontend
set REACT_APP_API_URL=http://[IP-ی-کۆمپیوتەرەکەت]:5000/api
npm start
```

### هەنگاوی سێیەم: کردنەوە لە مۆبایل

1. دڵنیابە کە مۆبایل و کۆمپیوتەر لە هەمان WiFi ن
2. لە مۆبایلەکەت Chrome یان Firefox بکەرەوە
3. بڕۆ بۆ: `http://[IP-ی-کۆمپیوتەرەکەت]:3000`
4. لە مینیۆی وێبگەڕەکە، هەڵبژێرە "Add to Home screen"
5. ئێستا وەک ئەپێکی ڕاستەقینە کار دەکات! 🎉

## چارەسەرکردنی کێشەکان

**کێشە: ناتوانم لە مۆبایل بیکەمەوە**
- دڵنیابە کە هەردووکیان لە هەمان WiFi ن
- Firewall ی Windows پشکنین بکە
- IP Address ڕاست بەکاربهێنە

**کێشە: ئایکۆنەکان دەرناکەون**
- فایلی `frontend/public/generate-icons.html` بکەرەوە
- ئایکۆنەکان دابەزێنە و بیانگوێزەرەوە بۆ `frontend/public/`
